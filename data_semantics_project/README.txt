"make_graph.py" takes in input the 3 .csv files and gives in output a .ttl file containing a RDF graph.
"make_questions.py" takes in input the .ttl file and gives in output some .json files.
Those .json files will be used to produce the games's questions.
A demo of the game is shown in "demo.mov".
The game gives in output a .csv file that will be used as input from "update_graph.py".
"update_graph.py" takes also in input the .ttl file and gives in output an updated version of it.
